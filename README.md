- `Cmd/Ctrl + Z`: Undo last action
- `Cmd/Ctrl + Shift + Z`: Redo last action
- `Cmd/Ctrl + S`: Format markdown list
- `Cmd/Ctrl + K`: Show keyboard shortcuts
- `Alt + â†‘/â†“`: Move line up/down
- `Tab/Shift + Tab`: Adjust indent level
